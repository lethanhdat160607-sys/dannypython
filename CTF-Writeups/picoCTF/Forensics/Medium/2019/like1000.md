## Code Bash Script
Initialization `(f=1000.tar)`: Sets the starting point from the outermost file.

Loop condition `(while [ -f "$f" ])`: The loop will continue running as long as the current compressed file exists in the directory.

Determine next target `(tar -tf ... | grep .tar)`: Uses the `tar -tf` command to "penetrate" the compressed file without emptying the data, then filters out the name of the next .tar file to assign to the next variable.

Decompress and Clean Up `(tar -xvf ... && rm "$f")`: Decompresses the current file. If successful, we immediately delete that compressed file to avoid cluttering the directory and save memory.

Update `(f=$next)`: Assigns the found file to the main variable to start a new decompression cycle.

Breakpoint `([ -z "$f" ])`: If there are no more .tar files inside the file (the Flag has been reached), the loop will automatically stop.

```
┌──(kali㉿kali)-[~/Tools]
└─$ f=1000.tar; while [ -f "$f" ]; do next=$(tar -tf "$f" | grep .tar); tar -xvf "$f" && rm "$f"; f=$next; done

```
